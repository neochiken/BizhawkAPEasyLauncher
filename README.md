Simple python script to make a .bat that auto opens bizhawk and bizhawkAPclient and connects them to help people play archipelago with their less tech savvy friends.

converted into exe using pyinstaller. [Download](https://github.com/neochiken/BizhawkAPEasyLauncher/releases)

This only works for games that use the generic bizhawk connector script, it wont work for ocarina of time or other games that have one specific to them, you can however force it to work by editing the final .bat and swapping the lua path with your needed lua script, though if you can do that you probably don't need this tool in the first place.

I have been informed that you can just rerun the patch file with ArchipelagoBizhawkClient.exe and it'll automatically do all this for you, however I took time programming this and learned a little bit of python so i'll leave it up anyways just incase anyone wants to look at it.
